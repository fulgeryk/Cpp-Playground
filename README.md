# C++ Playground

This repository is my personal C++ experimentation lab.

It exists as a space to explore the language deeply — from low-level language mechanics to algorithmic thinking and systems behavior. The goal here is not polished software, but **understanding**.

Many files in this repository are small experiments created to investigate specific aspects of C++, test assumptions, and build intuition about how the language behaves under the hood.

Some experiments are simple.
Some break things intentionally.
Some evolve into larger ideas.

That is part of the process.

---

## Areas of Exploration

### Core Language Mechanics

Experiments focused on understanding the foundations of C++:

* pointers and memory layout
* references and value semantics
* arrays and pointer arithmetic
* unions and type representation
* object lifetimes and RAII
* const correctness

These exercises focus on **what actually happens in memory**, not just syntax.

---

### Data Structures & Algorithms

Implementation of classical data structures and algorithmic patterns:

* linked lists
* binary search trees
* hash tables
* graph traversal (BFS / DFS)
* recursion and backtracking
* algorithmic problem solving

These experiments strengthen problem-solving skills and reinforce concepts often used in technical interviews.

---

### Memory & Performance

Since C++ gives direct control over hardware and memory, part of this repository explores performance-related topics:

* stack vs heap behavior
* memory allocation strategies
* cache behavior
* performance comparisons between implementations

---

## Learning Philosophy

This repository is not about copying solutions.

It is about:

* understanding *why* things work
* testing ideas through code
* debugging until the root cause becomes clear
* implementing concepts from scratch
* learning through experimentation

Some code will be messy.
Some experiments will fail.
Some ideas will be rewritten multiple times.

That is intentional.

---

## Current Focus

Current learning topics include:

* concepts from *The C++ Programming Language* by Bjarne Stroustrup
* modern C++ features and idioms
* algorithmic thinking and data structures
* deeper understanding of memory and performance

Each experiment in this repository reflects progress along that journey.

---

## Rule of This Repository

If something behaves unexpectedly, I investigate it.
If something breaks, I debug it.
If something is confusing, I study it until it becomes clear.

This repository is the record of that process.
